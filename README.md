# reactnative-boilerplate

This boilerplate includes the following:

- Expo
- React Native
- React Navigation
- Redux Toolkit
- tailwindcss

# Installation

```
npx degit rakshithp7/reactnative-boilerplate myapp
```

```
cd myapp
yarn install
```

or npm

```
cd myapp
npm install
```

# To start the server

```
expo start
```
